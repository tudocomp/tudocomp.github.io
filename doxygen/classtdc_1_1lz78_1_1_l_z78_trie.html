virtual node_t tdc::lz78::LZ78Trie< search_pos >::add_rootnode ( uliteral_t  c)
protectedpure virtual

The dictionary can store multiple root nodes For LZ78, we use a root node with the id = c = 0.

For LZW, we add for each possible literal value a root node. The compressor has to add these nodes.

Implemented in tdc::lz78::CedarTrie, tdc::lz78::MyHashTrie, tdc::lz78::TernaryTrie, tdc::lz78::BinaryTrie, tdc::lz78::BinarySortedTrie, and tdc::lz78::HashTrie.
