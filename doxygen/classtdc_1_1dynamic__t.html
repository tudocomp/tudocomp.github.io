Detailed Description

Custom integer type for storing values with a runtime-variable number of bits, with an maximum of 64 bits.

The bit with, however, is not actually stored in an instance of this type, so it behaves the same as a uint64_t in practice.

It exists to support the GenericIntVector<dynamic_t> specialization.
